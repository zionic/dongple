# LiveStatusBoard 컴포넌트 기능 고도화 작업 기록

## 1. 개요
* **작업 시점**: 2026-03-23
* **핵심 목적**: `LiveStatusBoard.tsx` 내 버튼 및 카드 기능을 동적이고 직관적인 사용자 참여(투표 및 제보)가 가능한 세분화된 UX로 발전시킴.
* **주요 작업 파일**: `src/components/dashboard/LiveStatusBoard.tsx`

---

## 2. 세부 기능 구현 내역

### 1) [맞아요] / [아니에요] 하이브리드 투표형 UI 도입
단순한 '인증하기'에서 벗어나, 데이터의 신뢰도 확보와 유저 친화적인 액션을 결합했습니다.
* **[👍 맞아요] 버튼 (원클릭 공감)**
  * 현재 표시된 상태(예: 여유, 혼잡)에 동의할 때 누름.
  * 클릭 시 추가 창 이동 없이 바로 인증 카운트(`verified_count`)만 1 증가시켜 직관성을 극대화.
* **[👎 아니에요] 버튼 (다른 상황 제보)**
  * 오프라인 상황이 기존 정보와 달라졌을 때 누름.
  * 클릭 시 **다른 상황 제보하기 모달**이 열려 새로운 상황 정보를 제출할 수 있는 분기 마련.

### 2) 상황 제보 모달 고도화 (뱃지 상태 선택기 도입)
* 최초 요청에 답변하는 `[지금 상황 알려주기]` 또는 상태를 갱신하는 `[아니에요]` 클릭 시 나오는 모달의 효율성 확보.
* 입력창과 별개로 **\`[🟢 여유] [🔵 보통] [🔴 혼잡]\`** 세 가지 직관적인 모드 선택 버튼을 추가.
* 선택 및 텍스트 작성 완료 후 제출하면 바로 메인 리스트의 요소가 해당 상태로 탈바꿈하며, 카운트는 최초 인증 1인으로 갱신됨.

### 3) 상태별 카드 컬러 배색 적용 (시각적 피드백)
시각적으로 훑어보는 것만으로 동네의 전반적인 분위기를 알 수 있도록 카드 렌더링 조건을 동적 업데이트함.
* 🟢 **여유**: 옅은 초록색 배경(bg-green-50/50), 짙은 초록 테두리 인디케이터
* 🔵 **보통**: 옅은 파란색 배경(bg-blue-50/50), 짙은 파란 테두리 인디케이터
* 🔴 **혼잡**: 옅은 빨간색 배경(bg-red-50/50), 짙은 빨간 테두리 인디케이터
* 🟠 **요청(답변 대기)**: 옅은 주황색 배경(bg-orange-50/50), 짙은 주황 테두리 인디케이터

### 4) '상세 공유 내용(코멘트)' 열람 모달 연동 완료
* 유저가 상태와 함께 작성해 제출한 상황 묘사 텍스트를 `LiveStatus` 배열 객체의 `message` 속성에 저장.
* **카드 본문(`div`) 클릭 시** 해당 구역의 코멘트 모달이 화면 위로 노출.
* 모달 화면 구성: 카테고리 뱃지, 장소 요약, 상태 결과, 인증 총계와 함께 큰 텍스트 뷰어 박스를 사용해 긴 텍스트의 이웃 피드백도 선명히 확인할 수 있도록 만듦.

## 3. 백엔드 연동을 위한 향후 태스크
현재 MVP로서 상태는 브라우저상(`useState` 배열 상태)에서만 시뮬레이션 중입니다. 다음 단계에서 완전한 연결을 해야 합니다.
* **기능 연결**: 열어둔 `src/services/statusService.ts` 파일의 로직과 Supabase를 활용.
* 1단계) 사용자가 모달을 통해 선택한 상태와 `message` 텍스트 필드를 DB `live_status` 테이블에 INSERT/UPDATE 하도록 적용.
* 2단계) [맞아요] 버튼 클릭 시 Supabase RPC 호출로 해당 row의 카운트를 증가시키는 로직 적용.

---

## 4. [더 많은 동네 상황 보기] UX 기획 확정 내역
버튼 클릭 시 처리 방안에 대해 논의한 결과, 위치 기반 서비스의 장점을 가장 훌륭히 살릴 수 있는 **1안(동네 지도 연동 뷰)** 방식으로 기획을 확정했습니다.

* **동작 방식**: 클릭 시 전용 지도 페이지(`/map` 등)로 전체 화면 전환
* **주요 UI 구성 (예정)**:
  * 전체 화면 지도 렌더링 (카카오맵 로컬 API 등 추가 활용)
  * 사용자들이 제보한 위치별 상황 핀(`여유`, `보통`, `혼잡`)이 지도 위에 마커로 즉시 노출됨
  * 하단 영역(Bottom Sheet)에 방금 개발한 예쁜 리스트뷰 카드가 썸네일 형태로 배치, 스와이프하여 전체 상황 탐색
* **기대 효과**: 직관성이 탁월하며, 유저들이 우리 동네 생활 지도를 실시간으로 꾸며나가는 재미와 참여도를 극대화할 수 있음.
