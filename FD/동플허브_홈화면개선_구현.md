나의 동플은 일기장이지만, 완전히 과거에 머물면 안 된다.
과거 기억 → 지금 상태 확인 → 다음 행동으로 이어져야 한다.

전체 UX 흐름
핵심 흐름 1
홈 → 지금 상태 확인 → 지도 → 공유
핵심 흐름 2
지도 → 마커 클릭 → 바텀시트 → 지금 공유 → 상태 반영
핵심 흐름 3
공유 → 자동 저장 → 나의 동플 → 다시 방문 판단
핵심 흐름 4
나의 동플 → 예전에 좋았던 곳 확인 → 현재 상태 보기 → 방문 결정
최종 화면 우선순위
화면	우선순위	이유
지도	1	동플의 핵심
공유	2	데이터 생성
홈	3	빠른 판단
나의 동플	4	재방문/리텐션
최종 원칙

홈은 판단을 만든다.
지도는 탐색을 만든다.
공유는 실시간 데이터를 만든다.
나의 동플은 다시 돌아올 이유를 만든다.

한 줄로 정리하면:

동플은 지금을 보고, 순간을 남기고, 다시 선택하게 만드는 서비스다.
